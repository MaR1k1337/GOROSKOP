# GOROSKOP 

A small educational web project: a main page with zodiac navigation and separate pages for each zodiac sign.

## Project structure
- `index.html` — main page with links to sign pages
- `1.html`, `2.html`, `3.html` etc — zodiac sign pages 
- `img/` — zodiac sign images (PNG/JPG/SVG)

## Features
- Simple static HTML pages
- Image support for each zodiac sign
- Team workflow: multiple contributors, branches, and merge
