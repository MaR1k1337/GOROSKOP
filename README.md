# 🌌 GOROSKOP 

A small but visually stunning educational web project that serves as an interactive guide to the zodiac signs. The project features a deep space aesthetic with modern CSS effects and animations.

## ✨ Features

* **Space UI:** Deep dark background with colorful nebula effects created using the `filter: blur()` CSS property.
* **Starry Sky Animation:** An integrated JavaScript generator that randomly creates hundreds of twinkling stars in the background using CSS `@keyframes`.
* **Glassmorphism:** Content cards feature a semi-transparent background with a frosted glass effect (`backdrop-filter`).
* **Responsive Grid:** The main menu is built with CSS Grid for perfect button alignment across different screens.
* **Git Workflow:** The project was developed using Git best practices, including branching, merging, and logical step-by-step commits.


🛠 Built With
HTML5 — Semantic page structure.

CSS3 — Styling, gradients, shadows, animations, Flexbox, and CSS Grid.

Vanilla JavaScript — Custom script for generating and animating the background stars.

Git & GitHub — Version control (forking, branching, and merging).

🚀 How to Run
This project consists of static files and does not require a local server or any additional dependencies.

Clone the repository to your local machine.

Open the index.html file in any modern web browser (Chrome, Firefox, Safari).
